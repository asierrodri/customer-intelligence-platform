# Customer Intelligence Platform

Plataforma de análisis de clientes orientada a entender churn y comportamiento de usuarios.

## Stack

- Python
- Pandas
- MySQL
- Docker
- ETL pipelines

## Estructura

- `etl/` → procesos de extracción y transformación
- `eda/` → análisis exploratorio
- `db/` → estructura de base de datos
- `docker/` → contenedores del proyecto

## Objetivo

Construir una pipeline reproducible de análisis de clientes que permita detectar patrones de churn.
